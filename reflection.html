Prelim

I learned the basics of photoshop editing. Fixing/Restoring images, making vectors, applying filters, using lasso tools, transforming photos, etc.

Activity 1: Computer Screen

In this activity, I learned how to put a picture on a computer screen by distorting the picture. I learned that you can
distort a picture through Transform → Distort. By distorting a picture, you can adjust the corners of the picture
and change the direction the picture is facing.

Activity 2: Out of Bounds

In this activity, I learned how to make an out of bounds effect on a picture of a girl on a swing. The first thing I did was
draw a rectangular marquee tool that will serve as the border of the picture later. After that, I put a layer mask on
the out of bounds parts, in this case, the hair of the girl. Then, I put a border by applying stroke on the rectangular
marquee that I previously made. Lastly, I added a background that suits the photo and put it at the bottom layer.

Activity 3: Drawing Effect

In this activity, I learned how to make a drawing effect of a picture. The first thing I did was duplicate the background
layer and change the blending mode of the copy to Color Dodge which is on top of the original background layer.
I then inverted the image by holding CTRL while pressing I. After that, I used Gaussian Blur until I can see the outlines of
the original image. You can find Gaussian Blur in Filter → Blur → Gaussian Blur. After that, I desaturated the image by adding
an adjustment layer and turning the saturation all the way down to -100. Finally, I adjusted the black and whites in the
image by adding an adjustment layer and playing with the levels.

Activity 4: Fire Text Effect

In this activity, I learned how to make a fire effect of a text. First, I made a text “Fire” on a black background. Then,
I adjusted the properties of the text which made it appear red. After that, I took a few fire images from another source
and put it on the text at the same time adjusting it so that it looks like the fire is burning on the text.

Activity 5: Tilt Shift Effect

In this activity, I learned how to put a tilt shift effect on a photo. I simply blurred the upper and lower part of the image
leaving the middle part untouched. I think this effect is appropriate on pictures of good scenery.

Activity 6: Glowing Text Effect

In this activity, I learned how to make a glowing effect on a text. This is similar to the fire text effect. The difference is I
adjusted the background this time. I chose the color green because I thought it will be soothing for my eyes but I was wrong.

Activity 7: Starburst Effect

In this activity, I learned how to make a starburst effect and a vector. I made the starburst effect by using only the gradients
and other filters. Later, I turned it to a vector.

Activity 8: Photo to Sketch Transition

In this activity, I learned how to make a transition from photo to sketch effect. First, I chose an appropriate background
for this effect which is a good scenery and a very noticeable subject, in this case, the shrine. After that, I went to
Filter → Stylize → Find Edges. And then, I desaturated the image by Image → Adjustments → Desaturate. Then, I put a layer
mask and added a black and white gradient over the image. This is the result. This image is taken from Thailand.

Activity 9: Fiery Portrait

In this activity, I learned how to put flames on a girl. I simply took a few fire from another source and put it around
the girl. And then, I carefully adjusted the properties of the flames such as its opacity and blending mode to fit the
image. This is a cool effect in my opinion.

Midterm

I learned how to use DreamWeaver to design webpages. I find it easy because the interface is easy to learn. There are also many useful tools.

Finals

I learned how to make flash animations using Adobe Flash. It is fairly easy to do if you are
patient. It's not hard to make animations, it just takes time to animate. But when you are
used to it, it will be a fun hobby. I even think of making animations and uploading them to
youtube.

My First Movie

In this activity, I learned the basics of flash; making use of different layers, creating
buttons, importing sounds, adding code snippets and using the basic tools.

Frame-by-Frame Animation

In this activity, I learned how to create a frame-by-frame animation. It is fairly easy
because all you need to do is insert keyframes and then move the object. Repeat it several
times and you are done!

Classic Tween

In this activity, I learned how to make an animation using classic tween. In my opinion, this
is the easiest way to animate a moving object.

Guided Classic Tween

In this activity, I learned how to create an animation using guided classic tween. I never knew the pencil tool has another use beside drawing.

Shape Tween

In this activity, I learned how to create an animation using shape tween. It's amazing how
you can just right click and select shape tween to transform an object to another. I'm
amaze. Much wow.

Basketball

In this activity, I learned how to make an animation in a rush.

Car Parking

In this activity, I applied what I've learned before. I think it was easy because I'm already
familiar with the tools.

Code Snippets

In this activity, I learned how to use code snippets. It is fairly easy to do because all you
need to do is select it from a list of actions and the code will automatically be generated.

Slideshow

In this activity, I made a slideshow of pictures from the prelim. I also learned how to create
fade effects by changing the alpha of the images. This is the activity I enjoyed the most.
